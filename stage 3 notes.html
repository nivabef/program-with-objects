3.2: Use Functions

Launching Python (Mac)

Click the magnifying glass icon and search for "IDLE." This opens the Python Shell. From here you can open an existing Python document or create a new Python document under the File menu.

Python Code to Open Website in a New Web Browser Window

import webbrowser
webbrowser.open("http://www.youtube.com")

"Webbrowser" is a module in the Python standard library and must be imported to use it.

Python Code to Suspend the Program for a Given Time

import time
time.sleep(10)

This counts in seconds, so the program suspends for 10 seconds in this example.
"Time" is module in the Python standard library and must be imported to use it.

Python Standard Library

The Python Standard Library is a list of the modules that are built into Python.

Abstraction

Abstraction is hiding of detail in programming. Example: We know we can use "webbrowser" to open a browser page, but the details of how it works are hidden.

This allows us to focus on the program we want to write.

os Module

Methods that can be called after importing the os module:

os.listdir("location of folder on computer")
Lists the files within the folder.

os.getcwd()
Gives you the folder the computer is currently working with (current working directory).

os.chdir("location of folder on computer")
Changes the CWD to the folder named in the parentheses.

os.rename(file_name, file_name.translate(None, "0123456789"))
In this example, the numbers in the file name would be removed and replaced with nothing.

3.3a: Use Classes- Draw Turtles

Classes

Classes are templates. They are a way to make copies of something without making multiple files.

Turtles

turtle.Turtle()
Here, "Turtle" is a class within the module "turtle."

brad=turtle.Turtle()
This creates in instance of the class Turtle.

Building Blueprint Analogy

A class is a blueprint of a building. It includes information about the height of the building, number of rooms, etc.

The same blueprint can be used to build multiple buildings that may differ by wall color, decorations, etc. The different buildings are instances of the blueprint. They are also called objects.

3.3b: Use Classes- Send Text

Twilio

In addition to the Python Standard Library, you can use other pre-made modules by downloading them. One of these is Twilio, which is a Python package that lets you send text messages using code.

Importing TwilioRestClient

from twilio.rest import TwilioRestClient

Inside Twilio there is a folder called "rest." Inside rest is a class called "TwilioRestClient." Using this code imports the class from the specific folder where it is located.

When we call the class TwilioRestClient in our program, we are really calling the __init__ function inside the class, which creates an instance of the class.

3.3c: Use Classes- Profanity Editor

Reading from a File

Code to pull text from an outside file and print it:

def read_text():
quotes = open("file location on computer")
contents_of_file = quotes.read()
print (contents_of_file)
quotes.close()

After the file is opened, the method "read" is used to read what's in it. When done with something you opened, you should always close it.

urllib Module

urllib lets you pull information from a website.

def check_profanity(text_to_check):
connection = urllib.urlopen(http://www.wdyl.com/profanity?q=" + text_to_check)
output = connection.read()
print(output)
connection.close()

In this code, the method urlopen opens a website that checks for profanity and returns a true or false value. It reads the text (text_to_check) that the program has read from an outside file and tells you whether or not it contains profanity.

3.4a: Make Classes- Movie Website

Google Python Style Guide

This guide provides conventions (how to name things, how to write comments, etc.) for programmers when writing programs.

Defining Classes

You can define your own classes (and normally save them in their own file that you will import) to use for your program.

class Movie():
def __init__(self, movie_title, movie_storyline, poster_image, trailer_youtube):
self.title = movie_title
self.storyline = movie_storyline
self.poster_image_url = poster_image
self.trailer_youtube = trailer_youtube

"Self" means that the class creates instances of itself. To use this class, you would write code that calls it to create a Movie object with the four given variables.

Definitions

Class - like a blueprint; can have both data and methods (ex: Movie)

Instances - different examples of a class (ex: Toy Story)

Constructor - called when we create an instance of a class; __init__ method is used inside the class; we initialize all of the data used within the instance

Self - the instance in question (ex: when Toy Story is being created, self is Toy Story)

Instance Variables - all the variables associated with a specific instance; unique to an object and can be accessed using the self keyword insude the class and the instance name outside the class

Instance Methods - all the functions inside the class that are associated with an instance and have the first argument as self

3.4a: Make Classes- Advanced Topic

Class Variables

Class variables are variables that all instances in a class share. You don't have to define them every time you create a new instance. The Google style guide says to use all CAPS for these.

__doc__, __name__ and __module__

__doc__
Calling this returns a document string that has been defined in a class.

__name__
Calling this returns the name of the class.

__module__
Calling this returns the name of the module in which the class was defined.

Inheritance

A child class inherits traits from a parent class. When you call the child class, it will in turn call the parent class and pull properties from it.
